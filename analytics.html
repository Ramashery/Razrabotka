      Установка трекера
      Трекер: /dc-tracker.js
      Данные пишутся в Firestore от всех посетителей.

      Автоматические события:
      • scroll_25pct / 50pct / 75pct / 90pct — глубина скролла
      • time_on_page_30s / 60s / 180s — время на странице
      • outbound_click — клик по внешней ссылке
      Кастомные события:
      dcTrack('click', { label: 'contact_button' });
dcTrack('view_project', { slug: 'my-project' });
dcTrack('form_submit', { form: 'contact' });
