      Данные от всех посетителей пишутся в Firestore.

      Коллекции:
      • analytics_sessions — сессии
      • analytics_pageviews — просмотры
      • analytics_events — события через dcTrack()
      Кастомные события:
      // Пример вызовов в main.js или на любой странице:
dcTrack('click', { label: 'contact_button' });
dcTrack('view_project', { slug: 'my-project' });
dcTrack('form_submit', { form: 'contact' });
